# Question Template

Use this template to ask questions about ZIME patent technology.

---

## Instructions

1. First check `FAQ.md` - your question may already be answered
2. Copy this template
3. Create a new file: `YYYY-MM-DD_topic.md`
4. Submit via pull request or issue

---

## Template

```markdown
# Question: [Your Question Title]

**Submitted by:** [Your Name/Handle]  
**Date:** [YYYY-MM-DD]  
**Category:** [Technical | Investment | Patent | Testing]

## Question

Write your question clearly and specifically.

## Context

Why are you asking this? What's your background or use case?

## What I've Already Checked

List any documents or resources you've already reviewed:
- [ ] FAQ.md
- [ ] EVIDENCE/ directory
- [ ] PROVISIONAL_2/SPECIFICATION.md
- [ ] Other: ___

## Related Claims

If your question relates to specific patent claims, list them here.
```

---

## Response Process

1. Questions are reviewed within 48 hours
2. Answers posted as replies in the same file
3. If broadly relevant, added to FAQ.md
